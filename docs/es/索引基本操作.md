> 发起请求工具：kibana

## 创建索引
- 索引名只能小写
- 索引名唯一
```
PUT /test_index
{
  "settings": {}
}
# settings：就是索引库设置，其中可以定义索引库的各种属性 比如分片数 副本数等，目前我们可以不设置，都走默认

PUT /test_index
{
  "settings": {}
}

```

## 判断索引是否存在
```
HEAD /test_index
```
